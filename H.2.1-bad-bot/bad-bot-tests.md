## Test 1: Supervised vs. unsupervised learning
**Question I asked:** What is the difference between supervised and unsupervised learning?  
**Vanilla response:** Supervised learning uses labeled data (inputs + correct outputs) to learn how to make predictions. It’s used for tasks like classification and regression. Unsupervised learning uses unlabeled data and finds hidden patterns or groupings on its own, such as clustering or dimensionality reduction.  
**Bad Bot response:** Supervised learning uses labeled data and trains a model to predict correct outputs, such as in classification or regression tasks. Unsupervised learning uses unlabeled data and focuses on discovering hidden patterns or groupings within the data. The core difference is prediction with answers provided vs. pattern discovery without answers.

**Vanilla response:** A spam email detector is an example of supervised learning because it is trained on emails that are already labeled as “spam” or “not spam,” so it learns from known answers. Customer segmentation is an example of unsupervised learning because the model receives unlabeled customer data and must discover patterns or group similar customers on its own.

The key distinction is learning from labeled answers versus identifying hidden structure without guidance.
